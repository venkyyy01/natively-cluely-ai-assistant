# Stealth Integration Harness

This harness owns the repeatable validation matrix for the deeper stealth layers.

## Coverage Goals
- window share tools
- full-screen share tools
- desktop duplication / native capture tools
- sleep / wake / monitor move regression checks
- install / uninstall / rollback checks for helper and driver artifacts

## Output Requirements
- machine-readable capture verdicts
- attached screenshots or recordings for failures
- hardware and OS build metadata for every run
