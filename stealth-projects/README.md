# Stealth Projects

This directory holds the scaffolds for the remaining stealth roadmap work that cannot be completed inside the main Electron app alone.

## Projects
- `contracts/` shared control-plane and telemetry contracts
- `macos-full-stealth-helper/` full stealth helper scaffold and v1 protocol contract
- `macos-virtual-display-helper/` `CGVirtualDisplay` helper scaffold
- `windows-idd-driver/` Indirect Display Driver scaffold and integration notes
- `windows-protected-render-host/` protected swap-chain render host scaffold
- `integration-harness/` repeatable capture-validation assets and notes

## Status
- Main app repo work is already integrated and feature-gated.
- These folders are starting points for the native projects that must be built, signed, and validated separately.
